Sistema de avaliação e gerenciamento - SAG

> O sistema consiste em uma plataforma educacional voltada para a gestão e aplicação de provas. O sistema deve cadastrar questões e gerar uma prova, deve ser capaz de cadastrar alunos, gerar e receber um gabarito. O sistema deve corrigir o gabarito dos alunos e salvar a nota do aluno no BD. A partir das notas dos alunos, o sistema precisa gerar um Dashboard e um relatório.
> 

> O sistema deve possibilitar vincular uma prova a uma instituição e uma turma.
> 

> O sistema deve gerar PDFs para as provas e gabaritos permitindo que o ator faça o download dos mesmo.
>

| Descrição | Os atores criam uma prova e adiciona questões. O ator deve sinalizar qual alternativa da questão está correta. |

## RF-008: Coletar gabarito

| Descrição | O sistema deve permitir que o **aplicador** envie os gabaritos preenchidos pelos alunos para processamento. O **QR Code** presente no gabarito será utilizado para identificar o aluno. A correção das respostas será realizada por um código em **Python**, que analisará a imagem do gabarito e identificará as alternativas marcadas. Após a análise, os resultados devem ser armazenados no banco de dados para cálculo das notas. |
| --- | --- |
| Ator | Admin, coordenador |
| Fluxo | 1. Botão para coletar gabaritos
2. Botão para abrir a câmera
2.1. O ator irá tirar uma foto do gabarito
3. Irá aparecer a resposta do aluno na tela
4. Botão para concluir
5. Mensagem de sucesso |
| Pré-condições | Autenticado no Sistema |
| Pós-condição | Voltar para página principal |
| Observação | **O ator poderá editar a reposta do aluno no topico 3.** |

# Requisitos não-funcionais

### RNF-001: Tecnologias

O sistema deve ser desenvolvido utilizando **PHP** no back-end e **React** no front-end, garantindo uma arquitetura moderna e escalável.

### RNF-002: Reponsividade

O sistema deve ser totalmente responsivo, garantindo uma experiência de uso adequada em dispositivos de diferentes tamanhos, como desktops, tablets e smartphones

### RNF-003: Dashboard

O sistema deve incluir um **dashboard interativo**, apresentando métricas e estatísticas sobre o desempenho dos alunos e turmas.

### RNF-004: Gerar PDFs

O sistema deve gerar provas e gabaritos no formato **PDF**

### RNF-005 QRcode

Cada gabarito deve conter um **QR Code**, permitindo rápida identificação do aluno e automação na correção.
